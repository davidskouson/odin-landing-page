Information about eating utensils.
You can use this device to cut food. It is called a knife.
You can use this device to eat solid food by poking it. It is called a fork.
You can use this device to eat food. It can be used for solid foods but holds liquids as well. It is called a spoon.
You can use this device to drink liquids. It is called a straw.
